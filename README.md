# NeuraLeap Website

## Overview

NeuraLeap is an advanced AI learning platform offering cutting-edge courses on large language models, neural networks, and AI safety. This repository contains the frontend code for the NeuraLeap website with Supabase integration for authentication.

## Features

* Responsive, modern UI using Tailwind CSS
* User authentication with Supabase
* Email verification
* Course catalog
* Student dashboard

## Technology Stack

* **Frontend**: HTML5, CSS3, JavaScript
* **CSS Framework**: Tailwind CSS
* **Authentication**: Supabase Auth
* **Database**: PostgreSQL (via Supabase)

## Local Development

To run this project locally:

1. Clone the repository
2. Serve the project using a local server
3. Open your browser and navigate to `http://localhost:8000`

## Supabase Configuration

The website uses Supabase for authentication with email verification. The Supabase project ID is configured in the JavaScript files.
